# DiscodeBot

- Discode内で使えるBotを作ってみる

# 環境

- pthon 3.9.1
- 外部ライブラリ：requirements.txtに記載
- TOKENやCLIENT_IDなどは.envファイルに記載し、constants.pyでまとめて呼び出し、変数に格納している
- Botのコードではconstants.pyから変数を呼び出し、TOKENなどを利用している
