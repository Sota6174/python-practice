# 実行時

1. ターミナルで```py test3.py```などファイルを実行する
2. Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)と言われるので、http://127.0.0.1:5000/ に飛ぶ
3. 開いたページがトップページを表すので、そこから実装通りにページを移動していく(例. http://127.0.0.1:5000/create)
